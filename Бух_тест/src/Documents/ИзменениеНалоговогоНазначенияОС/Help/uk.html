Документ "Зміна податкового призначення ОЗ"

Документ "Зміна податкового призначення ОЗ" використовується для зазначення нового податкового призначення з ПДВ, в якому буде задіяно основний засіб.

Оброблювані об'єкти вибираються з довідника "Основні засоби".

Для кожного об'єкта основних засобів вказується поточна балансова вартість у бухгалтерському і податковому обліку (з податку на прибуток)

Для більш швидкого заповнення табличної частини документа необхідно скористатися кнопкою "Підбір".

Після формування списку основних засобів необхідно скористатися кнопкою "Заповнити" для заповнення табличної частини інформацією про об'єкти основних засобів за даними обліку.

 

За фактом зміни податкового призначення ОЗ можуть коригуватися податкові зобов'язання.

Необхідність нарахування податкових зобов'язань (кредиту) вказується в полі Метод коригування, сума ПЗ (ПК) - в полі Сума ПДВ.
