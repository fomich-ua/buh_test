Призначений для зберігання параметрів ключових операцій.

Параметри ключових операцій необхідні для оцінки продуктивності роботи програми. У програмі передбачені вбудовані засоби для збору і аналізу даних про продуктивність роботи за APDEX, яка є міжнародним стандартом оцінки продуктивності інформаційних систем.

Список відкривається по команді Ключові операції розділу Сервісні підсистеми.

Для того щоб оцінювати продуктивність програми, необхідно на початку роботи програми заповнити параметри ключових операцій (після установки програми параметри ключових операцій не заповнені). У програмі є вбудовані ключові операції. При необхідності можна додати операції в список. Перераховані різні операції програми, наприклад, проведення документа, формування звіту і т. п. В списку виводяться відомості:

  • Найменування - повне найменування операції;
  • Пріоритет - ціле число, при цьому пріоритет не повинен повторюватися;
  • Цільовий час - час, витрачений на операцію в секундах;
  • Мінімально допустимий рівень - для порівняння взяті оцінки за apdex у цифровому вираженні, наприклад, Задовільно (apdex > = 0.75).

Вибрані параметри ключових операцій будуть застосовані при вимірах продуктивності.
