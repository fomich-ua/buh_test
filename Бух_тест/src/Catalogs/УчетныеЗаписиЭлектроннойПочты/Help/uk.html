Призначений для ведення облікових записів електронної пошти, які використовуються для отримання і відправки електронних повідомлень.

Список можна відкрити відповідною командою в розділі Адміністрування - Органайзер. 

У списку виводиться:

  • Найменування облікового запису;
  • Ім'я користувача - вказується ім'я відправника листів, воно може відрізнятися від імені користувача в програмі;
  • Адреса електронної пошти облікового запису у форматі name@server.net. 

Для облікових записів, отримання та відправка з яких виконується автоматично без участі користувача, обов'язково повинен бути введений пароль. Також для коректної роботи програми обов'язково повинна бути налаштована системний обліковий запис електронної пошти.

Системний обліковий запис електронної пошти

  • Програма поставляється з напередвизначеним обліковим записом - системної обліковим записом електронної пошти. По умовчанню використовується для розсилки різних повідомлень з програми, наприклад, для автоматичної відправки повідомлень адміністратора по завершенню оновлення програми. Перед використанням цих можливостей необхідно настроїти обліковий запис.
  • Настройка системного облікового запису провадиться по відповідній команді в розділі Настройка і адміністрування - Органайзер. Також можна Змінити її в списку.

Введення облікового запису електронної пошти

  • При спробі збереження програма перевіряє введені облікові записи електронної пошти (намагається відіслати пробне повідомлення). Облікові записи, які не пройшли перевірку, не зберігаються.
  • Натисніть кнопку Створити, введіть необхідні поля:
    • Вкажіть Адресу електронної пошти облікового запису у форматі name@server.net;
    • Введіть Пароль облікового запису. Для системних облікових записів рекомендується запам'ятати пароль, для того щоб програма могла розсилати повідомлення від такої облікового запису без участі користувача. Тим не менш, рішення про зберігання паролів в інформаційній базі приймається у кожному випадку індивідуально;
    • Встановіть з допомогою прапорців, як Використовувати обліковий запис:
      • Для відправки пошти;
      • Для отримання пошти;
    • Вкажіть Ім'я відправника (Ваше ім'я або назву організації);
    • За допомогою перемикача визначте спосіб створення облікового запису:
      • Визначити параметри підключення автоматично (рекомендується);
        • Натисніть Створити. Програма зробить спробу автоматичної настройки та перевірки облікового запису. Дотримуйтесь інструкцій програми. При цьому якщо при автоматичній настройці стався збій, можна спробувати налаштувати параметри вручну (див. нижче). 
      • Настроїти параметри підключення вручну;

    • Натисніть Далі для заповнення решти параметрів, при цьому програма проставляє рекомендовані параметри для більшості поштових серверів. Їх потрібно тільки перевірити і спробувати зберегти обліковий запис заново.

Настройка облікового запису електронної пошти

  • НатиснітьЗмінити, за допомогою програми виконайте необхідні настройки.

Перевірка облікового запису електронної пошти

  • Виділіть обліковий запис у списку, натисніть Ще - Перевірити обліковий запис для перевірки введених параметрів облікового запису. При цьому відправляється стандартне поштове відправлення і перевіряється можливість з'єднання з сервером вхідних поштових повідомлень (якщо обліковий запис використовується для отримання пошти).
  • Якщо для облікового запису не прописаний пароль, програма його запитує. При необхідності введіть пароль.
