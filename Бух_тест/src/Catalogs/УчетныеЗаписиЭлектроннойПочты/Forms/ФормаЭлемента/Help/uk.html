Опис властивостей облікового запису електронної пошти

При введенні облікового запису більшість параметрів проставляється програмою по умовчанню в залежності від настройок поштових серверів облікового запису електронної пошти.

Для системних облікових записів рекомендується запам'ятати пароль, для того щоб програма могла розсилати повідомлення від такої облікового запису без участі користувача. Тим не менш, рішення про зберігання паролів в інформаційній базі приймається у кожному випадку індивідуально.

Основні відомості

  • Введіть Найменування облікового запису. Необхідно сформулювати найменування так, щоб воно коротко характеризувало обліковий запис, відображало призначення облікового запису або її власника, наприклад, "Служба техпідтримки". Саме найменування буде з'являтися в різних списках.
  • Введіть Адресу електронної пошти облікового запису в форматі користувач@поштовийсервер.
  • Вкажіть Ім'я відправника листів - ім'я власника облікового запису, яке буде вказуватися в кореспонденції, при цьому воно може відрізнятися від імені користувача в програмі, наприклад, користувач "Адміністратор", а ім'я його "Іван Іванович".
  • Вкажіть, як Використовувати обліковий запис:
  • Увімкніть прапорець Для відправки листів;
  • Увімкніть прапорець Для отримання листів.

Відправка листів (настройка сервера вихідної пошти)

  • Вкажіть Адресу електронної пошти облікового запису в форматі користувач@поштовийсервер, він може відрізнятися від основного, в такому разі він буде призначений тільки для відправки листів.
  • При необхідності введіть Пароль облікового запису (якщо він не введений, програма запитує його при відправці листів).
  • При необхідності увімкніть прапорець Безпечна перевірка пароля.
  • У полі Сервер вкажіть адресу SMTP-сервера, наприклад, "smtp.ukr.net".
  • Вкажіть Порт комп'ютера для відправки поштових повідомлень, наприклад, "465".
  • Увімкніть прапорець Використовувати безпечне з'єднання (SSL).
  • При необхідності увімкніть прапорець Необхідний вхід на сервер перед відправкою.

Отримання листів (настройка сервера вхідної пошти)

  • Вкажіть Адресу електронної пошти облікового запису в форматі користувач@поштовийсервер, він може відрізнятися від основного, в такому разі він буде призначений тільки для отримання листів.
  • При необхідності введіть Пароль облікового запису (якщо він не введений, програма запитує його для отримання листів).
  • При необхідності увімкніть прапорець Безпечна перевірка пароля.
  • У полі Сервер вкажіть адресу POP3 сервера, наприклад, "pop.ukr.net". 
  • Виберіть Протокол сервера вхідної пошти: POP або IMAP.
  • Вкажіть Порт комп'ютера для отримання вхідної пошти, наприклад, 993.
  • Увімкніть прапорець Захищене з'єднання (SSL).
  • Увімкніть прапорець Залишати копії повідомлень на сервері, для того щоб програма не вилучила листи в папці "Вхідні" поштової скриньки електронної пошти. По умовчанню прапорець відключений, в цьому випадку всі листи після отримання будуть вилучені з сервера.
  • Визначте термін зберігання листів на поштовому сервері за допомогою поля Вилучати листи з сервера через - наприклад, через "10 днів".

Інші настройки

  • Вкажіть Тривалість очікування сервера - наприклад, "30 сек".

Перевірка настройок

  • Натисніть Перевірка настройок для перевірки введених параметрів облікового запису.

  • Програма відправляє стандартне поштове повідомлення і перевіряє можливість з'єднання з сервером вхідних поштових повідомлень (якщо обліковий запис використовується для отримання пошти). Також можна скористатися однойменною командою меню Всі дії.
