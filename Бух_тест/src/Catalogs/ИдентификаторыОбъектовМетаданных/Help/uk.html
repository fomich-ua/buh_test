Призначений для зберігання і перегляду ідентифікаторів об'єктів метаданих. Дані списку не редагуються, призначені для забезпечення роботи програми.

Список заповнюється автоматично при першому запуску (і оновленні) конфігурації, згідно метаданих конфігурації, а також автоматично актуалізується при кожному оновленні версії конфігурації (враховуються перейменовані, додані і віддалені об'єкти метаданих).

У списку у вигляді дерева виводяться всі об'єкти програми, згруповані за типами. За замовчуванням виводиться поле Синонім об'єкта. По команді Всі дії - Змінити форму можна додати в список поля Повне ім'я та Ідентифікатор.

Натисніть для перегляду докладних відомостей про ідентифікатор вибраного об'єкта.
