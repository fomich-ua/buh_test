Картка файлу містить різноманітні відомості про файл і його поточній версії і кнопки для виконання основних дій з файлами. Додаткові команди для роботи з файлом знаходяться в меню Всі дії.

Для відкриття картки файлу натисніть клавішу F2 абоЗмінити в списку файлів. Картку можна також відкрити зі списку приєднаних файлів об'єкта програми, повнотекстового пошуку або із звітів.

Відомості про файл

  • Найменування - повне найменування файлу.
  • Опис - докладний опис файлу.
  • Створений - час і дата додавання файлу в базу.
  • Автор - користувач, який додав файл.
  • Редагує - користувач, який зайняв файл для редагування.
  • з - дата і час початку редагування файлу.
  • Папка - папка, в якій збережено файл. За допомогою кнопки можна вибрати іншу папку.
  • Код - код файла.

Поточна версія

  • Номер - номер версії файлу.
  • Автор - користувач, який додав версію файлу.
  • Створена - дата додавання версії файлу в базу.
  • Дата зміни - дата зміни даних версії файлу (місцевий час).
  • Розмір (байт) - розмір даних версії файлу.
  • Розширення - формат файлу, поруч значок, який також показує формат.

Зберігання версій

  • Увімкніть прапорець Зберігати версії, для того щоб при завершенні редагування створювалася нова версія файлу. При знятому прапорці оновлюється файл в сховище. Рекомендується зберігати версії, т. к. при псуванні файлу під час редагування можна відновити файл з його версії. По умовчанню прапорець включений.

При використанні ЕП і шифрування стають доступними вкладки:

Електронні підписи

На вкладці розміщуються відомості про електронний підпис файлу:

  • Автор підпису - користувач, який підписав файл;
  • Дата підпису - дата підписання файлу;
  • Коментар - напишіть додаткові відомості про підписи;
  • Статус - істинність підписи.

За допомогою відповідних команд можна:

  • Підписати файл;
  • Відкрити підпис - відкрити вікно з основними властивостями сертифіката ЕП виділеного файлу: кому і ким виданий, призначення і термін дії сертифіката, на основі якого створена підпис;
  • Перевірити, чи змінювався ким-небудь підписаний файл, за допомогою одного або всіх сертифікатів ЕП. Для цього виділіть один або кілька підписів. Якщо ЕП файлу, що зберігається, не відповідає якому-небудь сертифікату ЕП, то такий ЕП після перевірки буде виділено в списку червоним кольором, а в колонці Статус у такого ЕП буде зазначено Не вірна;
  • Перевірити все - перевірити всі підписи;
  • Зберегти ЕП у файл на диск;
  • Вилучити у файла ЕП.

Зашифрований для

  • На вкладці розміщуються відомості про шифрування файлу:
  • У списку відображаються на цій вкладці сертифікати, за допомогою яких зашифрований файл. Подвійним клацанням миші або за допомогою відповідної команди можна Відкрити сертифікат ЕП і переглянути його основні властивості: кому і ким виданий, призначення і термін дії сертифіката.

З картки файлу можна перейти до перегляду і редагування пов'язаних з файлом дані:

  • версії файлів;
  • завдання.

Див. також:

  • Додаткові реквізити та відомості; 
  • Настройка форм;
  • Робота з формами.
