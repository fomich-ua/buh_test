Список призначений для управління файлами програми.

У списку відображаються відомості:

  •  - підписаний ЕП;
  •  - зашифрований;
  • Найменування - повне найменування файлу. Значок поруч з найменуванням вказує тип файлу;
  • Власник - папка списку файлів або об'єкт програми, до якого приєднаний файл;
  • Автор - користувач, який додав файл;
  • Створений - час і дата додавання файлу в базу;
  • Розширення - розширення файлу;
  • Розмір (Кб) - розмір файлу;
  • Редагує - користувач, який зайняв файл для редагування;
  • Опис - докладний опис файлу;
  • Версія - номер версії файлу;
  • Відредагував - користувач, який додав версію файлу.
  • Том - том на диску, в якому зберігаються файли;
  • Шлях до файлу на томі;
  • Дата - дата додавання версії файлу в базу.

Доступні дії:

  • Переглянути - Відкрити файл зовнішнім додатком для перегляду;
  • Відмінити редагування - зняти з файлу позначку зайнятості для редагування;
  •  Змінити - відкрити картку файлу;
  • Зберегти як - зберегти файл у вказаний каталог на жорсткому або мережевому диску.
  • Вилучити (Shift + Del) - видалити файл остаточно.
