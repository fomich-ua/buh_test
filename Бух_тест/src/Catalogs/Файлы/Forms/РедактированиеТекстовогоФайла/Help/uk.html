У вбудованому редакторі текстових файлів можна переглянути і відредагувати файл, змінити кодування, відкрити дерево процесів і завдань, в яких задіяний файл, порівняти з вихідним варіантом в процесі редагування, при необхідності відкрити за допомогою стандартного додатка.

Відкривається зі списку Файли по команді Відкрити або Редагувати файл, у разі якщо у персональних настройках програми є відповідна настройка.

По умовчанню у вбудованому редакторі відкриваються текстові файли з розширенням TXT, INI, XML. Ви можете доповнити список розширень файлів у персональних настройках програми. На закладці Робота з файлами в таблиці Настройки відкриття файлів можна вказати Розширення файлів для роботи у вбудованому редакторі (перераховуються через пробіл). Також можна вказати Спосіб відкриття - Стандартною прикладною програмою або У вбудованому редакторі (установка по умовчанню).

Редагування файлу

  • Якщо файл відкритий в режимі перегляду, кнопки Записати і закрити і Записати недоступні. Натисніть Редагувати, для того щоб змінити файл.
  • Принцип роботи вбудованого редактора текстових файлів такий самий, як і у зовнішнього застосування. Файл копіюється на локальний комп'ютер і позначається як зайнятий в інформаційній базі. До тих пір поки ви не натиснете Всі дії - Закінчити редагування, створена версія файлу не буде доступна іншим користувачам. 
  • Після закінчення редагування файлу натисніть Всі дії - Закінчити редагування, для того щоб зняти позначку зайнятості для редагування. В залежності від настройок команда також зберігає версію файлу або оновлює файл в сховище. По умовчанню створюється нова версія файлу. Для зміни настройки при створенні версії зніміть відповідний прапорець.

Перегляд завдань

З допомогою команди Всі дії - Завдання можна переглянути список завдань, які були створені на підставі даного файлу.

Перегляд картки файлу

Натисніть Всі дії - Відкрити картку, для того щоб переглянути відомості про файл.

Порівняння версій

Натисніть Всі дії - Показати відмінності, для того щоб виконатипорівняння версій файлів. Якщо програма для порівняння версій не була обрана заздалегідь у персональних настройках програми (вкладка Робота з файлами), пропонується її вибрати. Камера Microsoft Word або OpenOffice.org Writer.

Збереження файлу

Натисніть Всі дії - Зберегти як, для того щоб зберегти файл у вказаний каталог на жорсткому або мережевому диску 

Відкриття файлу в стандартній програмі

  • Натисніть Всі дії - Відкрити стандартною прикладною програмою, для того щоб перейти на зовнішню програму (Блокнот Windows) для редагування файлу.
  • Якщо необхідно, щоб текстові файли c певними розширеннями по умовчанню відкривалися у зовнішньому додатку, внесіть зміни у персональні настройки програми. Для цього:
    • У персональних настройках програми перейдіть на закладку Робота з файлами.
    • Змініть Настройки відкриття файлів. У колонці Спосіб відкриття виберіть Стандартною прикладною програмою.

Зміна кодування текстового файлу

Текстові файли можуть бути збережені в різних кодуваннях. Основними кодуваннями для роботи з текстовими файлами російською мовою є:

  • IBM866 (Кирилиця DOS),
  • KOI8-R (Кирилиця KOI8-R),
  • Windows-1251 (Кирилиця Windows.

Якщо після відкриття текстового файлу замість тексту видно набір безглуздих символів, виконайте команду Всі дії - змінити кодування, виберіть іншу систему кодування зі списку.
