У цьому списку при завершенні роботи відображаються файли, які зайняті Вами для редагування.

Для того щоб скасувати відкриття списку зайнятих файлів при завершенні роботи, вимкніть прапорець Показувати зайняті файли при завершенні роботи. Також це можна зробити у персональних настройках програми на закладці Робота з файлами.

У списку відображаються відомості:

  • Найменування - найменування файла;
  • Опис - докладні відомості про файл (з картки файлу).

Доступні дії:

  • Відкрити файл - відкриває файл зовнішнім додатком для редагування. Також можна відкрити файл подвійним натисненням кнопки миші або за допомогою клавіші Enter.
  • Закінчити редагування - зберігає версію файлу і звільняє файл для інших користувачів.
  • Відкрити картку - відкрити для перегляду відомостей про файл.

Див. також:

  • Загальна інформація про файли,
  • Робота з формами.
