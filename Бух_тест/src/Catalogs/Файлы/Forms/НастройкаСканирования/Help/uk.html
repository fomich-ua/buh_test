Програма дозволяє зберігати файли прямо зі сканера. Для роботи зі сканером у програмі необхідно зробити настройку сканування.

Відкривається з вікна персональних настройок програми (сторінка Робота з файлами) по відповідній команді.

Перед виконанням настройок необхідно підключити до комп'ютера сканер, при цьому в залежності від можливостей підключеного сканера будуть доступні ті чи інші параметри. Для підтвердження настройок натисніть ОК.

Компонент сканування

  • Перед настройкою сканування натисніть Встановити компоненту сканування.
  • Якщо на Вашому комп'ютері немає сканера, встановлювати компоненту не потрібно.
  • Якщо компонента вже встановлена, то замість цієї кнопки відображається Версія встановленої компоненти сканування.

Ім'я сканера

У полі Ім'я сканера виберіть використовується сканер зі списку доступних пристроїв, підключених до комп'ютера.

Можливість перетворення PDF

  • Увімкніть прапорець, для того щоб Використовувати ImageMagick для перетворення в PDF. У цьому випадку відскановані зображення можна перетворити у формат PDF з допомогою програми ImageMagick. На клієнтському комп'ютері, до якого підключено сканер, повинні бути встановлені безкоштовні програми ImageMagick (використовуйте варіант Win32 dynamic at 8 bits-per-pixel) і Ghostscript.
  • Введіть Шлях до програми перетворення в PDF - шлях до виконуваного файлу програми ImageMagick (по умовчанню "convert.exe"). Якщо це ім'я конфліктує з іншими програмами за допомогою кнопки Вибрати потрібно вказати повний шлях виду "C:\Program Files\ImageMagick-6.6.3-Q8\convert.exe".

Параметри зображення

  • Формат односторінкового зображення - дозволяє вибрати один з підтримуваних форматів (BMP, JPG, GIF, PNG, TIF, PDF) для поміщення файлу в інформаційну базу. Якщо прапорець Використовувати ImageMagick для перетворення PDF включений, то в списку стає доступним також формат PDF (в цьому випадку можна вибрати PDF для зберігання, інші формати - для сканування).
    • Якість - якщо вибраний формат JPG, то можна налаштувати його якість (100 - максимальна якість і максимальний розмір, 1 - мінімальна якість і мінімальний розмір). По умовчанню - 100.
    • Стиснення - якщо вибраний формат TIF, можна настроїти стиснення. Можливі варіанти: LZW, RLE, CCITT3, CCITT4, Без стиснення. По умовчанню: Без стиснення.
    • Тип відсканованого зображення - дозволяє вибрати один з підтримуваних форматів (BMP, JPG, GIF, PNG, TIF) для зображень, які будуть отримані зі сканера. Опція доступна, якщо обраний формат односторінкового зображення PDF. У цьому випадку формат PDF використовується для зберігання.
  • Формат багатосторінкового зображення - параметр доступний, якщо прапорець Використовувати ImageMagick для перетворення PDF включений. Дозволяє вибрати один з підтримуваних форматів (TIF, PDF) для поміщення багатосторінкового файлу в інформаційну базу. Застосовується, коли при роботі з Помічником сканування декількох відсканованих зображень обрана команда Прийняти всі як один файл.

Параметри сканування

Залежать від можливостей вибраного сканера, який підключений до програми. Деякі з них можуть бути недоступні.

  • Виберіть Розрізняючу здатністьсканування (DPI). Список доступних розрізняючих здатностей залежить від обраного сканера.
  • Виберіть Кольоровість зображень, які будуть отримані зі сканера (Монохромне, Градації Сірого, Кольорове). Залежить від можливостей сканера.
  • Виберіть один з варіантів Повороту зображення після сканування. Для деяких моделей сканерів ця властивість недоступна.
  • Виберіть Розмір паперу, яка поміщена в сканер. Для деяких моделей сканерів ця властивість недоступна.
  • Увімкніть прапорець Двостороннє сканування. У цьому випадку будуть відскановані обидві сторони паперу. Для деяких моделей сканерів ця властивість недоступна.

Діалог сканера

Показувати діалог сканера перед скануванням - якщо прапорець включений, перед скануванням буде показаний діалог для розширеної настройки сканування (наприклад, вибір області сканування на аркуші паперу). По умовчанню прапорець включений.

Установка параметрів по умовчанню

Натисніть Встановити стандартні параметри для встановлення настройок сканера за замовчуванням. Якщо програм перетворення у формат PDF на комп'ютері немає, слід залишити формати, рекомендовані програмою по умовчанню.

Примітка: при роботі під керуванням ОС Windows Server 2008 для використання сканера вам потрібно встановити компоненту Можливості робочого столу (Desktop Experience).
