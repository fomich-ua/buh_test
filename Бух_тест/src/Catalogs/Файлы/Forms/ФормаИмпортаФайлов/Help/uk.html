Призначена для завантаження файлів з диска на комп'ютері.

Відкривається зі списку файлів по команді Завантажити файли контекстного меню по правій кнопці миші після вибору файлу (файлів) у вікні Провідника Windows. У провіднику виберіть один або кілька файлів, натисніть Відкрити.

Виводиться список завантажуваних файлів. Можна замінити весь список з допомогою кнопки Вибрати (доповнити перелік неможливо, при виборі нових файлів старі вилучаються) - відкрити Провідник Windows і вибрати інші файли.

Заповніть необхідні для завантаження файлів відомості:

  • У полі Папка автоматично проставляється вибрана папка в списку файлів, в яку буде завантажені файли з диска на вашому комп'ютері. При необхідності можна вибрати іншу папку;
  • Для економії місця на диску увімкніть прапорець Вилучати файли після додавання (по умовчанню вимкнено);
  • Напишіть Опис для файлів, що створюються;
  • Увімкніть прапорець, для того щоб Зберігати версії (рекомендується, по умовчанню прапорець включений);
  • Для текстових файлів можна Вибрати кодування зі списку.
  • Для підтвердження натисніть Завантажити файл.
Також можна використовувати перетягування мишею для завантаження файлів з жорсткого або мережного диска - просто перетягніть файли мишкою з Провідника Windows у потрібну папку списку файлів програми.
