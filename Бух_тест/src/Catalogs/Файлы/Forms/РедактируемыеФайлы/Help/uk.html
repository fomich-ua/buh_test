У цьому списку відображаються лише файли, зайняті Вами для редагування. Редагувати файли можна в списку файлів або в списку приєднаних файлів об'єктів програми.

Відкривається по команді Файли, що редагуються панелі навігації розділу Органайзер.

У списку відображаються відомості:

  • Файл - найменування файлу
  • Де лежить - папка, в якій знаходиться файл або об'єкт, до якого приєднаний файл.

Доступні дії:

  • Відкрити файл - Відкрити файл для перегляду;
  • Закінчити редагування - зберігає версію файлу і звільняє файл для інших користувачів.
  • Відкрити картку - відкрити для перегляду відомостей про файл.

Див. також:

  • Загальна інформація про файли,
  • Робота з формами.
