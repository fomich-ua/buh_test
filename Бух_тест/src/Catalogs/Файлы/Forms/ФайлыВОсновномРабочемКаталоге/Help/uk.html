Основний робочий каталог - спеціальний каталог на Вашому комп'ютері, у якому програма розміщує переглядаються і редаговані Вами файли. Список призначений для управління файлами в робочому каталозі.

Викликається з персональних настройок програми (закладка Робота з файлами) по команді Настройка основного робочого каталогу - Відкрити список файлів.

У списку виводиться:

  • Файл - найменування файла;
  • Розмір (Кб);
  • Дата зміни;
  • На редагуванні - прапорець включений, якщо файл, що редагується.

З допомогою команди Всі дії - Змінити форму можна додати в список додаткові поля.

Доступні дії

  • Для видалення непотрібних файлів з робочого каталогу натисніть Вилучити. При цьому будуть вилучені тільки ті файли, які Вами не вносилися будь-які зміни.
  • Для того щоб звільнити зайняті Вами файли, використовуйте команду Скасувати редагування.
  • Для того щоб зберегти зміни у файлі і звільнити його для інших користувачів, натисніть Закінчити редагування.
  • Натисніть Відкрити каталог файлу, для того щоб переглянути теку, в якій зберігається редагований файл.
  • Натисніть , щоб відкрити картку файлу.
