Перегляд відомостей про версію файлу.

Після закінчення редагування файлу (по команді Завершити редагування у списку файлів) програма рекомендує зберегти версію файлу і ввести інформацію про дану версію. По умовчанню цю можливість включено (рекомендується використовувати збереження версій, так як у випадку серйозних помилок, які призвели до псування файлу, можна повернутися до попередньої, незіпсованої версії). Згодом можна відредагувати коментар і при необхідності змінити автора версії.

Відомості про версію файлу

  • Власник - найменування файла-власника, автоматично заповнюється при додаванні версії, не підлягає редагуванню;
  • Найменування - найменування версії, автоматично заповнюється при додаванні версії, не підлягає редагуванню;
  • Батьківська версія - для того, щоб підпорядкувати одну версію інший, виберіть зі списку потрібну версію. Історія версій у програмі для наочності представляється у вигляді дерева;
  • Розмір (байт) - розмір даних версії файлу в байтах.
  • Розширення - розширення (формат) файлу. Поруч розташовується картинка, яка також показує формат файлу, визначається програмою автоматично.
  • Коментар до версії - введіть додаткові відомості про версію файлу (також можна ввести при збереженні версії по закінченні редагування файлу).
  • Автор - користувач, який додав версію файлу. При необхідності можна Вибрати іншого автора зі списку користувачів.
  • Дата створення - дата та час додавання файлів в базу.
  • Номер версії - нумерація версій ведеться програмою автоматично при змінах файлу, при цьому максимальний номер має остання збережена версія.
  • Дата зміни - дата зміни даних версії файлу (місцевий час).
  • Кодування - автоматично вибирається значення По умовчанню.

Перегляд версії файлу

Натисніть , щоб відкрити вибрану версію файлу відповідним додатком для перегляду або редагування (також можна відкрити файл подвійним клацанням миші по потрібному рядку або по команді Всі дії - Відкрити).

Збереження файлів на диск

Для того щоб зберегти файл на диск або інший пристрій, натисніть або виконайте команду Всі дії - Зберегти як. Виберіть каталог для збереження файлу.
