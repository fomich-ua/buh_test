Вибір програми для порівняння версій файлів. У програмі передбачена можливість порівнювати версії файлів формату HTML, TXT, RTF, DOC, DOCX, ODT.

Дозволяє задати текстовий редактор, який буде використовуватися для порівняння версій файлів між собою. Викликається при виконанні команди Показати відмінності зі списку версій файлу, якщо текстовий редактор для порівняння не заданий за замовчуванням.

Вибір способу порівняння версій

У полі Порівнювати версії файлів за допомогою виберіть програму для порівняння файлів зі списку: для вибору можливі Microcoft Word або OpenOffice.org Writer.

Установка настройки по умовчанню

Для того щоб не вибирати текстовий редактор кожний раз при порівнянні версій, необхідно заповнити настройки по умовчанню. Для цього у персональних настройках програми (вкладка Робота з файлами) в полі Порівнювати версії за допомогою необхідно задати текстовий редактор для порівняння версій. для вибору можливі Microcoft Word або OpenOffice.org Writer. Ця настройка по умовчанню не встановлюється автоматично.
