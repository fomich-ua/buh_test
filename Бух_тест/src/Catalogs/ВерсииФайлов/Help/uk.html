Призначений для перегляду і управління версіями файлу.

Для перегляду списку всіх версій файлу відкрийте його картку (у списку файлів натисніть F2 або Відкрити картку) і натисніть Версії файлів на панелі навігації.

Список версій є ієрархічним, для зручності перегляду відображається у вигляді дерева. У цьому випадку легко простежити історію редагування файлу. Більш пізні версії відображаються підпорядкованими більш раннім. При цьому першу версію файлу видалити зі списку неможливо.

Після закінчення редагування файлу програма рекомендує зберегти версію файлу і ввести інформацію про дану версію. По умовчанню цю можливість включено (рекомендується використовувати збереження версій, так як у випадку серйозних помилок, які призвели до псування файлу, можна повернутися до попередньої, незіпсованої версії). Зберігання версій можна включити за допомогою відповідного прапорця в картці файлу або при завантаженні файлів (папок). Згодом можна змінити деякі відомості про версії, наприклад, коментар.

У списку виводиться:

  • Автор - користувач, який додав версію файлу. Також в цій колонці у вигляді картинки відображається тип файлу.
  • Номер версії - нумерація версій ведеться програмою автоматично при змінах файлу, при цьому максимальний номер має остання збережена версія.
  • Коментар - додаткова інформація про версію файлу.
  • Дата створення - дата та час додавання файлів в базу.
  • Розмір (байт) - розмір даних версії файлу в байтах.

З допомогою команди Всі дії - Змінити форму можна додати додаткові поля, наприклад:

  • Повне найменування файла - повне ім'я файлу;
  • Тип зберігання файлу - приймає значення В інформаційній базі або В томах.
  • Розширення - формат файлу, і деякі інші поля.

Перегляд версії файлу

Натисніть Відкрити версію, для того щоб відкрити вибрану версію файлу відповідним додатком для перегляду або редагування (також можна відкрити файл подвійним клацанням миші по потрібному рядку або за допомогою команди Всі дії - Відкрити версію).

Перегляд відомостей про версію файлу

Натисніть Відкрити картку, для того щоб відкрити відомості про обрану версію. У картці версії можна змінити Коментар.

Заміна поточної версії файлу

Оберіть потрібну версію файлу в списку, натисніть Зробити активною, для того щоб змінити активну (поточну) версію. При цьому нумерація версій залишається незмінною, змінюється лише порядок версій у списку.

Порівняння двох версій файлу

Виділіть дві версії у списку і натисніть кнопку Показати відмінності для їх порівняння. Порівняння версій підтримується для наступних типів файлів:  HTML, TXT, RTF, DOC, DOCX, ODT.

Збереження файлів на диск

Для того щоб зберегти файл на диск або інший пристрій, натисніть або виконайте команду Всі дії - Зберегти як. Виберіть каталог для збереження файлу.
