Опис властивостей поля контактної інформації.

Поставка програми здійснюється c напередвизначеними полями контактної інформації для деяких документів і списків. У визначених полів контактної інформації можна тільки поміняти настройки, інші реквізити заповнені по умовчанню і захищені від редагування.

Типи контактної інформації

Програма дозволяє зберігати наступні типи контактної інформації:

  • Адреса - поле з таким типом контактної інформації призначене для зберігання адреси у вигляді довільного рядка або у вигляді шаблону адреси. Напередвизначені фактична та юридична адреси використовуються в друкованих формах документів.
  • Телефон - поле з таким типом контактної інформації призначене для зберігання телефонних номерів у вигляді довільної рядка, або у відповідності з шаблоном введення номера телефону. Напередвизначений телефон використовується в друкованих формах документів.
  • Адреса електронної пошти - поле з таким типом контактної інформації призначене для зберігання адреси електронної пошти. Реквізит використовується при відправленні електронних листів з програми.
  • Веб-сторінка - поле з таким типом контактної інформації призначене для зберігання адрес веб сторінок.
  • Факс - поле з таким типом контактної інформації призначене для зберігання номерів факсів у вигляді довільної рядка, або у відповідності з шаблоном введення номера факсу. Реквізит використовується в друкованих формах документів.
  • Інше - поле з таким типом контактної інформації може використовуватися для зберігання довільної текстової контактної інформації, наприклад, можна зберегти інформацію про те, як дістатися до організації. Програма дозволяє в це поле введіть текст до 1024 символів.

Додавання поля контактної інформації в список або документ

Заповніть необхідну інформацію:

  • Виберіть зі списку Тип контактної інформації.
  • Введіть Найменування, з яким поле контактної інформації з'явиться у списку або документі.
  • Введіть текст Підказки, яка буде відображатися при наведенні курсору на поле контактної інформації.
  • Виберіть Групу, якщо вона не введена по умовчанню. Група відповідає документу або списку програми, в який додається нове поле контактної інформації.

Настройки полів контактної інформації

  • Увімкніть прапорець Обов'язкове заповнення, щоб програма перевіряла заповнення поля контактної інформації при введенні документів або списків. Збереження даних без заповнення цього поля буде неможливо.
  • Увімкніть прапорець Дозволити введення декількох значень, для того щоб можна було ввести декілька значень. Наприклад, в організації або фізичної особи може бути кілька адрес електронної пошти або номерів телефонів, за якими можна зв'язатися з ними.
  • Настройка Редагувати лише в діалозі використовується для типів контактної інформації Адреса, Телефон, Факс. Така настройка дозволяє обмежити введення нової і редагування існуючої адреси, телефону або факсу шаблоном введення. При встановленому прапорці передбачена можливість ввести адресу у вільній формі. Це може знадобитися у виняткових випадках, наприклад, якщо дані адреси в будь-яких документах наводяться з помилкою, але для друку необхідно їх залишити. При цьому така адреса буде зберігатися в програмі з позначкою, що вона введена у вільній формі, і використовуватися з обмеженнями. Після уточнення адресу можна буде виправити, заповнивши за шаблоном.

Особливості настройки поля Адреса 

  •  Увімкніть прапорець Тільки український, для того щоб заборонити вибір іншої країни при заповненні адреси. Використовуйте цю можливість, якщо у співробітників Вашої організації тільки українські адреси, це прискорить введення інформації. При цьому стають доступними додаткові настройки:
    • Приховувати неактуальні адреси - увімкніть прапорець, для того щоб програма не показувала в списках застарілі адреси на момент останнього завантаження класифікатора адреси. Такі адреси помічені в списках значком . За замовчуванням прапорець відключений.
    • Включати країну до представлення - увімкніть прапорець, для того щоб при автоматичному складанні повної адреси в поле Адреса вказувалася країна.

Особливості настройки поля Адреса електронної пошти

Якщо встановлено тип поля Адреса електронної пошти, то програма перевіряє правильність вводу адреси електронної пошти. Увімкніть прапорець Забороняти введення некоректних адрес, в цьому випадку ввести некоректну адресу електронної пошти буде неможливо.
