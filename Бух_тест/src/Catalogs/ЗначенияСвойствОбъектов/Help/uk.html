У списку задаються значення додаткових реквізитів і відомостей.

У програмі передбачена можливість використовувати додаткові реквізити (відомості) декількох типів. Тип даних вибирається при настройці додаткових реквізитів (відомостей). При виборі типу даних Додаткове значення програма пропонує ввести значення в цей список. При цьому для кожного додаткового реквізиту або відомості даного типу використовується свій список значень. За допомогою цього типу даних можна описувати різні властивості об'єктів програми.

Тип даних Додаткове значення часто використовується, при цьому введення інформації зводиться до вибору значень із списку, що прискорює процес введення і знижує кількість помилок.

Застосовується як звичайний список значень, так і згрупований.

Вибір значень

  • Виберіть потрібний елемент.
  • Натисніть Вибрати.
  • Також можна скористатися подвійним клацанням миші.

Введення значень

Для введення даних натисніть кнопку Створити, заповніть необхідні поля.

Групування значень

Якщо значень багато, список можна згрупувати. Для цього введіть необхідну кількість груп. Групи можуть існувати незалежно один від одного або підкорятися один одному.

  • Для введення групи в список натисніть Створити групу, заповніть необхідні поля. 
  • Перетягніть значення мишею у відповідні групи.
