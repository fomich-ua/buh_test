Призначена для збереження варіанту звіту.

Після того, як звіт настроєний і сформований , його можна зберегти в новому варіанті звіту. Для цього необхідно скористатися командою Всі дії - Зберегти варіант.

Список існуючих варіантів

  • При збереженні варіанту звіту виводиться список всіх існуючих варіантів цього звіту - напередвизначених і користувацьких.
    • Список включає:
      • стандартні, передбачені в програмі напередвизначениі варіанти звітів. По умовчанню є спільними;
      • спільні варіанти звітів, які надані іншими користувачами у спільний доступ;
      • особисті варіанти звітів, які збережені раніше.
    • У списку також виводяться відмічені на вилученя варіанти звітів.
  • У списку можна Змінити існуючі варіанти звітів за допомогою відповідної команди контекстного меню по правій кнопці миші. У напередвизначеного варіанту звіту можна змінити опис або настройки розміщення, у користувацького варіанту можна змінити і найменування (при наявності відповідних прав).
  • Для того щоб вилучити існуючий користувацький варіант звіту, виберіть його, натисніть на клавішу Del або виконати команду контекстного меню по правій кнопці миші Відмітити для вилучення. Напередвизначені варіанти звітів вилучити неможливо.

Найменування варіанту звіту

  • Для збереження нового варіанту звіту необхідно ввести Найменування, яке не зайняте іншим варіантом цього звіту. При збереженні контролюється унікальність найменувань варіантів звітів.
  • При перезапису варіанти можна зберегти настройки в існуючому варіанті - по умовчанню вибрано варіант звіту, в якому була виконана команда Зберегти варіант. Для перезапису можна вибрати будь-який особистий варіант звіту зі списку. Для варіантів звітів, які не можна замінити, в тому числі і напередвизначених, програма у найменуванні по умовчанню додає слово "-копія". У разі якщо замінний варіант звіту був відмічений для вилучення, програмою виводиться попередження. Натисніть кнопку Так, щоб підтвердити свій вибір.
    • Для адміністраторів програми або відповідальних за звіти передбачена можливість перезапису загальних варіантів звітів інших авторів.

Доступність

  • За допомогою перемикача Доступний адміністратор варіанту звіту може обмежити доступність варіанту звіту - вибрати доступність Тільки для автора.
  • Якщо вибрати доступність Всім користувачам, то можна вказати видимість для інших виконавців. Для того, щоб варіант звіту був відразу видно в панелях звітів всіх користувачів, увімкніть прапорець По умовчанню видно в панелі звітів.

Збереження варіанта звіту

Зберегти варіант звіту можна двома способами:

  • Після визначення основних параметрів натисніть кнопку Зберегти, при цьому будуть використані настройки по умовчанню і новий варіант звіту буде поміщений в той самий розділ списку звітів, що і вихідний варіант.
  • Натисніть Далі, для того щоб змінити настройки розміщення варіанту звіту. 

Розміщення в панелях звітів

  • Увімкніть один або кілька прапорців у колонці Розділ, група для розміщення варіанту звітів у відповідних розділах або групах програми. При розміщенні в розділі варіант звіту буде виводитися у верхній частині панелі звітів розділу, без розміщення у групах.
  • У колонці Важливість виберіть спосіб виділення варіанту звіту в групі:
    • Важливий – варіант звіту буде виділено жирним шрифтом.
    • Див. також – варіант звіту буде виведений в окремій групі внизу панелі звітів.
  • Введіть Опис, яке виводиться під найменуванням звіту в панелі звітів та в списках звітів.
  • Натисніть кнопку Зберегти для збереження варіанта звіту з вибраними настройками.
