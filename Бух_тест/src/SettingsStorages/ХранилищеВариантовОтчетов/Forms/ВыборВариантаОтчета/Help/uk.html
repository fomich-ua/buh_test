Призначений для вибору варіантів звітів.

При роботі зі звітом можна швидко завантажити раніше збережений варіант звіту. Для цього натисніть Вибрати варіант або Всі дії - Вибрати варіант.

Для вибору доступні:

  • стандартні, передбачені в програмі зумовлені варіанти звітів. По умовчанню є спільними і в списку відображаються зі значком ;
  • спільні варіанти звітів, надані іншими користувачами в спільний доступ. У списку вони відображаються зі значком . Для вибору доступні варіанти всіх авторів;
  • особисті варіанти звітів, які збережені раніше.

Список вибору може містити три групи варіантів звітів:

  • використовувані варіанти звітів (не виділяються в окрему групу, розташовані на початку списку); 
  • Приховані в панелях звітів - у цю групу входять напередвизначені спільні звіти, у яких вимкнений прапорець По умовчанню видно в панелі звітів;
  • Відмічені для вилучення. Група завжди розміщена внизу списку вибору. При виборі відміченого для вилучення варіанта програма виводить попередження.

У списку виводиться:

  • Найменування - повне найменування варіанту звіту, яке відображається в списках.
  • КартинкаДоступний всім користувачам:
    • видно - варіант звіту доступний всім користувачам,
    • не видно - варіант звіту доступний тільки автору.
  • Автор - повне ім'я користувача, який є автором варіанту звіту.
  • У нижній частині списку виводиться Опис - детальна інформація про вибраний варіант звіту.

Вибір варіанту звіту

  • Для того щоб відкрити варіант звіту, виділіть його в списку, натисніть Вибрати.
  • Також можна скористатися подвійним натисненням кнопки миші.

Відбір

Якщо список варіантів звітів великий, то з допомогою відбору за автору можна швидко знайти потрібний варіант звіту.

  • У полі Автор виберіть автора варіанту звіту зі списку користувачів.

Настройка варіанту звіту

  • Виберіть варіант звіту, натиснітьЗмінити, змініть необхідні поля.

 Вилучення варіанту звіту

  • Користувацький варіант звіту з допомогою команди меню Всі дії можна Відмітити для вилучення. Також можна скористатися клавішею Del.
  • Повторно виконайте цю команду, для того щоб Зняти відмітку вилучення. 
  • Ця можливість недоступна для напередвизначених варіантів звітів.

Адміністратору/Відповідальної за звіти:

Зазвичай особисті варіанти звітів інших авторів приховані. Але в рідкісних випадках може знадобитися прямо з цього списку вибору зробити один з особистих варіантів звітів загальнодоступним або призначити його іншому користувачеві. Для того щоб побачити всі варіанти звітів без винятку, натисніть Всі дії - Показати особисті варіанти звітів інших авторів.
