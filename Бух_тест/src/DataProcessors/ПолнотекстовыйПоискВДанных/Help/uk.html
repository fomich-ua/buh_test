Повнотекстовий пошук даних дозволяє знаходити будь-які відомості не лише по полях карток, але і по змісту приєднаних файлів.

У поле введення рядок пошуку можна вставити навігаційне посилання на дані. У цьому випадку пошук зроблений не буде, а буде відразу відкрита потрібна картка. Навігаційну посилання на будь-який об'єкт інформаційної бази можна отримати за допомогою команди у заголовку кожного вікна 1С:Підприємства 8.

У файловому варіанті інформаційної бази індекс повнотекстового пошуку не оновлюється автоматично і тому повинен періодично оновлюватися вручну. Якщо індекс повнотекстового пошуку не актуальне (тобто такі дані, які ще не пройшли індексування), то у файловому варіанті інформаційної бази стає доступна кнопка «Оновити індекс».

Поради:

  • Якщо знайшлося багато даних, то уточніть пошуковий запит, додавши до нього якесь уточнення (дату, суму, назву контрагента) 
  • Якщо знайшлося мало даних, то зробіть запит, виключивши з нього яке-небудь слово
  • Для точного пошуку використовуйте:
  • Лапки - для пошуку точної фрази, наприклад: «"міні пилосос"»
  • Для пошуку одного з двох слів, наприклад «Акація АБО Стройпромвсе» - знайде всі згадки контрагентів ТОВ "Акація" та ЗАТ "Стройпромвсе".
  • Дужки - для групування виразів, наприклад, пошук «Іванов (Акація АБО Стройпромвсе)» знайде всі документи, виписані Івановим по контрагентам ТОВ "Акація" та ЗАТ "Стройпромвсе".
  • Для пошуку з урахуванням синонімів використовуйте "!", наприклад, пошук "!номер" знайде всі документи, в яких зустрічається слово "номер" або "число"
  • Для "нечіткого" пошуку використовуйте "#" наприклад "Стройпромвсе#2" знайде всі документи, що містять такі слова, які відрізняються від слова "Стройпромвсе" на одну або дві букви.
  • Для пошуку по початку слова використовуйте "*", наприклад, пошук "Строй*" знайде всі документи, які містять слова, що починаються на "Лад" - "Будівництво", "Стройпромвсе" і т. д.
