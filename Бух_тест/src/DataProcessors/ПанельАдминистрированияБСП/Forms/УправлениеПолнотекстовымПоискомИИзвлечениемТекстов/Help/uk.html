Повнотекстовий пошук використовується для пошуку даних в програмі по вмісту довідників, документів і інших об'єктів програми.
Для початку роботи з повнотекстовим пошуком необхідно увімкнути прапорець Повнотекстовий пошук даних.
Після цього з'явиться можливість управляти індексом повнотекстового пошуку, оновлювати і очищати його.

Команди управління повнотекстовим пошуком:

  • Оновити індекс
    		•  Виконує оновлення повнотекстового індексу, додаючи до нього нові або змінені об'єкти.
  • Очистити індекс
    		•  Вилучає всі файли повнотекстового індексу.

    Автоматичне витягування текстів

    Витягування текстів з файлів необхідно для повнотекстового пошуку в програмі.
    У клієнт-серверному варіанті якщо один або декілька робочих процесів сервера працюють під Linux, а частина або всі клієнти підключаються з допомогою веб-клієнта, то у інформаційній базі можуть бути додані файли, текст яких не витягнутий, і відповідно ці файли не можуть бути знайдені повнотекстовим пошуком по вмісту.
    У цьому випадку на одному з клієнтських комп'ютерів, що працюють під управлінням ОС Microsoft Windows потрібно запустити тонкий або товстий клієнт, і на ньому відкрити обробку «Автоматичне витягування текстів». Ця обробка в автоматичному режимі буде витягувати текст з файлів.
    По умовчанню інтервал часу виконання дорівнює 60 секундам.
    У файловому варіанті роботи тексти витягуються автоматично при завантаженні файлу в програму.
  • Редагувати регламентне завдання
    		•  Розклад та інші параметри регламентного завдання по витягуванню текстів з файлів.
  • Витягування текстів
    		•  Запуск автоматичного витягування текстів з файлів на робочої станції під управлінням ОС Windows.
