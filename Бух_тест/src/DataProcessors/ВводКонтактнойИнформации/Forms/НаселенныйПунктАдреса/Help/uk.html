Призначена для вводу населеного пункту за детальним шаблоном.

Відкривається з шаблону введення адреси в поле Місто, населений пункт по кнопціВибрати.

Поля Область, Район, Місто, Нас.пункт призначені для введення відповідних частин адреси. Поля, які повинні бути заповнені, залежать від адреси. Наприклад, в деяких адресах може не бути району, в інших - міста або населеного пункту. Підтвердіть свій вибір, натиснувши кнопку Так для продовження вводу.

Після закінчення введення міста або населеного пункту натисніть ОК для продовження вводу адреси.
