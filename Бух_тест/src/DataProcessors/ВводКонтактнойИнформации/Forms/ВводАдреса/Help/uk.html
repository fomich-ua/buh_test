Призначена для вводу адреси по шаблону.

  • У полі Країна виберіть країну адреси. Редагування адреси за межами України, провадиться в загальному полі, без урахування частин адреси. Країна "Україна" вибрана програмою по умовчанню, при цьому поле може бути недоступним в залежності від настройок. Для того, щоб змінити країну, натиснітьВибрати. Вибір здійснюється із списку Країн світу.
  • Поле Індекс призначене для введення поштового індексу, відповідного даній адресі.
  • Поле Місто, населений пункт призначено для введення регіону, міста і т. п. При необхідності можна ввести частині адреси детально, натиснувшиВибрати. У відкритому шаблоні введіть дані населеного пункту.
  • Поле Вулиця призначене введення вулиці адреси.
  • Поля Будинок, Корпус, Квартира дозволяють ввести відповідні частини адреси. Можна змінити ці параметри, вибравши додаткові значення з випадаючого списку.
  •  Також можнаДодати додаткові поля для введення інших напередвизначених частин адреси, таких як Будова, Літера, Споруда, Ділянка, Офіс, Бокс, Приміщення, Кімната. 
  • Поле Представлення адреси при заповненні адреси за шаблоном недоступне, заповнюється програмою автоматично в міру заповнення поля адреси.
  • Поле Коментар дозволяє вказати довільний коментар до адреси, який не буде включатися в її представлення і виводиться на друк.

Приклади заповнення адреси

  • Для введення адреси "Київ, Московський проспект, будинок 2, будова 2, офіс 12345" необхідно:
    • У поле Місто, населений пункт вибрати "Київ".
    • У полі Вулиця вибрати "Московський проспект".
    • У полі Дім написати "2".
    • Для введення номера будови необхідно Додати поле, вибрати з випадаючого списку значення Будова і проставити в полі значення "2".
    • Замість поля Квартира вибрати з випадаючого списку поле Офіс і ввести значення "12345".

Введення адреси у вільній формі

  • Скористайтесь цією командою меню Всі дії (в інтерфейсі Таксі Ще), для того щоб при введенні адреси за шаблоном ввести свідомо неправильну адресу (наприклад, з товаром на накладній неправильну адресу постачальника, його треба тимчасово зафіксувати для друку, але надалі його використовувати не можна). Адреса використовується для зберігання, тому можна рекомендувати використовувати його тільки для поодиноких випадків.
  • Виводиться шаблон для введення адреси, але всі його поля, крім поля Представлення адреси і Коментар, стають неактивними. Заповніть поле Представлення адреси. При переході до поля Коментар або натисканні кнопки ОК заповнюються ті поля шаблону, які програмі вдалося розпізнати в рядку адреси (тобто в цих полях немає помилок). Перевірка заповнення адреси в даному випадку відключена.
  • Для підтвердження адреси натисніть ОК. Програмою буде зафіксовано, що адреса заповнений у вільній формі і на перегляд адресу буде виводитися в такому вигляді.

Очищення адреси

З допомогою команди меню Всі дії (в інтерфейсі Таксі натисніть Ще) - Очистити адресу можна стерти неправильно введену адресу.
