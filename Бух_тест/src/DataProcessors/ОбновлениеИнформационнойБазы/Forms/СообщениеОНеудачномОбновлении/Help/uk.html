Призначено для інформування про помилку під вільний час оновлення ІБ.

Для того щоб подивитися в журналі реєстрації відомості про помилки, перейдіть по гіперпосиланню Подробиці див. у Журналі реєстрації. 

Для того щоб вийти з програми, натисніть кнопку Завершити.

Для того щоб спробувати запустити програму ще раз, натисніть Перезапустити.
