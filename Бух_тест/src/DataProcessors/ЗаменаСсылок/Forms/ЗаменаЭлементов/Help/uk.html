Дозволяє замінити один або кілька елементів на вибраний елемент.

Така можливість затребувана при вилученні елементів, наприклад, так як для остаточного видалення елемента необхідно позбутися посилань на нього.

Командою Замінити у всіх місцях використання можна скористатися з вікна Вилучення відмічених об'єктів.

Можна автоматично замінити всі посилання на елементи, які вилучаються, посиланнями на вибраний елемент. Всі зазначені елементи будуть відмічені на вилучення або вилучені безпосередньо, якщо користувач володіє адміністративними правами.

Заміна елементів

Виводяться всі елементи цього списку, у тому числі відмічені на вилучення.

  • Натисніть для перегляду обраного елемента.
  • Для документів передбачений відбір по періоду. Якщо документів у списку багато, можна скористатися цим відбором. Для цього натисніть і Встановіть інтервал дат для перегляду.
  • Виберіть елемент, на який необхідно замінити виділені елементи.
  • Внизу списку в інформаційному рядку зазначаються реквізити замінних елементів і реквізити елемента, який обраний для заміни. Також для користувачів з адміністративними правами надається можливість вибору. Перейдіть за допомогою гіперпосилання та вкажіть, що замінні елементи:
    • відмічені на вилучення;
    • вилучені безповоротно.
  • Натисніть Замінити для заміни всіх посилань.

У разі виникнення помилок в процесі обробки елементів буде виведений список необроблених об'єктів з описом причини неможливості обробки. У цьому випадку можна:

  • Відкрити елемент для перегляду, натиснувши або подвійним натисканням миші;
  • Спробувати ще раз замінити елемент, натиснувши Повторити заміну;
  • Натиснути Назад і підібрати для заміни інший елемент;
  • Натиснути Закрити для повернення в програму.
