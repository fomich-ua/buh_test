Призначена для реєстрації змін даних у програмі. Це необхідно для подальшої відправки даних при синхронізації в інші програми.

Список можна відкрити по команді Всі дії - Склад даних, що відправляються зі списку Синхронізації даних.

Ведеться реєстрація номерів повідомлень. Натисніть на гіперпосилання з номерами повідомлень, щоб їх переглянути або виправити.

Можливий відбір змін - виберіть зі списку:

  • Всі;
  • Тільки вивантажені;
  • Тільки невивантажені.

Список складається з двох частин:

  • ліва частина складається з двох колонок:
    • Об'єкт конфігурації - перераховані об'єкти програми (довідники, документи тощо) у вигляді дерева;
    • Зареєстровано/не вивантажено - кількість змінених (або нововведених) і не вивантажених елементів у цих об'єктах;
  • права частина змінюється в залежності від виділеного об'єкта в лівій частині:
    • По виділеній групі об'єктів у колонці Опис наводиться докладний опис статистичних даних лівої частини.
    • По виділеному об'єкту права частина програми поділяється на дві колонки:
      • Найменування конкретного об'єкта (наприклад, Валюта) - наводиться перелік усіх змінених і не вивантажених елементів списку;
      • Номер відправленого - номер повідомлення, в якому відправлені змінені елементи списку (вказується номер повідомлення або виводиться Не вивантажувалося);

Реєстрація змін та скасування реєстрації

Змінені і знову вводяться об'єкти програма реєструє автоматично. Для ручної реєстрації об'єктів виберіть об'єкт у дереві за допомогою прапорців, натисніть Зареєструвати зміни: 

  • - усіх об'єктів вибраних типів - можна вибрати об'єкт і всі його підпорядковані об'єкти;
  • - об'єктів вибраних типів з авторегістраціею;
  • - одиночного об'єкта (виберіть із списку об'єкт або створіть новий);
  • - об'єктів за допомогою відбору (вкажіть відбір, відберіть потрібні елементи);
  • - результатів запиту.

Для того щоб скасувати надсилання всіх даних, приготованих для інших програм, в дереві об'єктів виділіть корінь (інформаційну базу) і натисніть Скасувати реєстрацію змін - Всіх об'єктів вибраних типів) - всі настройки аналогічні попереднім.

Інші можливі дії:

  • Для того щоб настроїти параметри реєстрації змін об'єктів, натисніть Настройка. Можна настроїти параметри автореєстрації, встановити номер повідомлення при реєстрації, при необхідності вказати шлях до каталогу консолі запитів.
  • Для того щоб переглянути результат вивантаження, виділіть об'єкт, за яким зареєстровано зміни, в лівій частині списку, при цьому в правій частині з'являється список елементів, виберіть потрібний елемент, натисніть Показати результат вивантаження або Всі дії - Стандартний результат вивантаження.
