Призначена для відбору даних, які будуть відправлені при синхронізації даних в іншу програму.

Відкрити список можна натиснувши на гіперпосилання у полі З відбором на сторінці Дані для відправлення Помічника інтерактивної синхронізації даних. 

Відбір даних для відправлення

  • Натисніть Підібрати, для того щоб вибрати дані для відправлення і заповнити список. Об'єкти програми для вибору виводяться у вигляді дерева. Виділіть натисніть Вибрати. Дані розміщуються в списку для відправлення.
  • Для видалення даних зі списку, натисніть Вилучити.
  • Список відправлених даних складається з колонок:
    • Дані для відправлення - вибрані для відправлення об'єкти програми перераховуються в цій колонці;
    • Відбір - По умовчанню вибрані Всі об'єкти (гіперпосилання). Натисніть на гіперпосилання, для того щоб відібрати потрібні елементи вибраних списків:
      • Натисніть Додати умову, виберіть Поле для відбору, у колонці Вид порівняння виберіть необхідну операцію порівняння, вкажіть Значення. При необхідності умов для відбору може бути декілька, в цьому випадку вони групуються за допомогою логічних операцій И, Или, Не по команді Згрупувати умови;
    • До відправлення/всього - підраховується кількість обраних елементів і загальна кількість елементів списків. Натисніть на гіперпосилання Буде відправлено внизу списку, для того щоб перерахувати вибрані елементи.
  • Натисніть OK для підтвердження свого вибору і повернення у Помічник для продовження настройки синхронізації даних.

Перегляд складу вивантаження даних

  • Натисніть Склад даних, для того щоб переглянути Відомості про склад даних, що відправляються.

Збереження настройок вивантаження та їх використання

За допомогою кнопки Настройки можна зберегти введені настройки для відбору даних або використовувати раніше збережені.

  • Програма пропонує використовувати попередні настройки, наприклад збережену автоматично, для цього виберіть із запропонованого списку значення Останнє відправлення.
  • Можна вибрати із списку раніше збережену настройку.
  • Можна зберегти свої настройки. Для цього натисніть кнопку Зберегти поточну настройку. Напишіть найменування настройки, що зберігається, натисніть Зберегти.

Підрахунок кількості надісланих об'єктів

  • У нижній частині списку В полі Буде відправлено натисніть на гіперпосилання, для того щоб розрахувати кількість відправлених об'єктів. Після підрахунку в колонці списку До відправлення/Всього відповідно проставляється кількість елементів і гіперпосилання приймає значення підсумкової суми відправляються елементів.
