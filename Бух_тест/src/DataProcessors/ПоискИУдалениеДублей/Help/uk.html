Призначена для пошуку і усунення дублювання елементів у всіх списках програми (для адміністраторів програми).

При роботі у програмі користувачів часто помилково вводяться дублі (наприклад, неправильно проводився пошук елемента, в результаті його не знайшли і ввели заново), які заважають правильному обліку. В документах вказують один екземпляр, то інший, в результаті при проведенні документів виникають неправильні проводки, і баланс не сходиться. Тому від дублювання в програмі необхідно позбавлятися.

Програма дозволяє знайти входження всіх повторюваних елементів і виробляє в документах заміну всіх посилань на дублі посиланнями на вибраний "правильний" елемент, потім відмічає на вилучення повторювані елементи списку.

Команду Пошук і усунення дублів можна знайти в розділі Адміністрування - Підтримка та обслуговування.

Пошук дублів

  • У полі шукати в виберіть один із списків програми.
  • У полі Відбирати вкажіть умови відбору елементів списку, для цього перейдіть за гіперпосиланням Всі елементи (значення по умовчанню) і визначте Правила відбору.
    • Виберіть Поле, по якому виробляти відбір і значення для пошуку в полі Вид порівняння виберіть операцію для порівняння і задайте Значення для порівняння. Натисніть ОК для повернення до пошуку дублів. Гіперпосилання прийме значення умови відбору.
    • Для вилучення відбору натисніть
  • У полі Порівнювати гіперпосилання автоматично проставляються напередвизначені Правила пошуку дублів в обраному списку. Для додавання елементів до правил натисніть гіперпосилання, прапорцями вкажіть елементи правила, натисніть Вибрати. Після вибору гіперпосилання змінить значення.
  • По закінченні складання умови пошуку дублів натисніть пошук.
  • Якщо знайти дублі не вдалося, то можна поміняти умови і натиснути Повторити пошук.

Список дублів

  • Список знайдених дублів складається з двох частин:
    • У лівій частині - знайдені елементи. Показуються в списку у вигляді дерева. Групуються по найменуванню.
    • У правій частині видається інформація по виділених елементів:
      • по групі елементів (тобто по кожному найменуванню) - звіт про кількість знайдених дублів в виділеного елемента.
      • за елементами - звіт про використання в програмі (Фраза "Не використовується" або перелік документів програми, в якому використаний цей елемент.
  • Один з елементів, які використовуються в програмі, автоматично вибирається як Оригінал.
  • Для того щоб змінити елемент, який необхідно залишити в програмі, виберіть його, натисніть кнопку Відзначити як оригінал.
  • Дублі у списку зазначаються прапорцями. 
  • Натисніть Відкрити для того, щоб переглянути виділений елемент. Також можна скористатися подвійним клацанням миші.
  • За допомогою кнопок Розгорнути групу дублів або Згорнути групу дублів можна керувати видимістю груп елементів у списку.
  • В інформаційному рядку внизу списку вказується кількість елементів і кількість знайдених дублів.
  • Якщо список знайдених дублів не влаштовує, можна змінити умови пошуку та натиснути Повторити пошук.

Вилучення дублів

  • Якщо всі перераховані в інформаційній рядку списку умови влаштовують, натисніть Вилучити дублі. Дублі будуть відмічені на вилучення, всі їх входження в документах замінюються на вибраний Оригінал.
  • Для виходу без вилучення натисніть Закрити. Умови пошуку та список дублів не зберігаються.
  • Остаточно вилучити дублі можна по команді Вилучення відмічених об'єктів розділу Адміністрування - Підтримка та обслуговування.

Перегляд посилань на елементи списку

За гіперпосиланням Всі місця використання можна перейти до перегляду звіту Місця використання по елементам, що входять в список.
