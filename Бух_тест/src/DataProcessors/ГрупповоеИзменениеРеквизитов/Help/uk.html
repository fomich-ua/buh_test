Групова зміна реквізитів

Групова зміна реквізитів передбачено у двох варіантах:

  • в більшості списків програми передбачена команда Змінити виділені.
  • тільки для адміністратора: у розділі програми Адміністрування - Підтримка та обслуговування по команді Групова зміна реквізитів або у разі підключення в якості зовнішньої обробки - у відповідному розділі програми по команді Додаткові обробки - Групова зміна реквізитів.

Зміна виділених елементів списку 

Зміна значень реквізитів (а також додаткових реквізитів і відомостей) у вибраних елементів списку.

Введення нових значень реквізитів вибраних елементів списку

  • Доступні для зміни поля обраних елементів списку відображаються у рядках колонки Реквізит. Можна змінити відразу кілька реквізитів.
  • Для того щоб змінити значення реквізитів (а також додаткових реквізитів і відомостей, потрібних рядках необхідно заповнити Нове значення.
  • Відкрийте рядок для введення за допомогою подвійного натиснення миші. Можна:
    • ввести значення для пошуку в списку;
    • з допомогою команди Показати всі (клавіша F8) вибрати значення з відповідного списку;
    • ввести нове значення в список за допомогою команди Створити.
  • При введенні нового значення автоматично включається прапорець .
  • Натисніть Змінити реквізити.

Очищення значень полів вибраних елементів списку

  • Для очищення значення реквізиту потрібно увімкнути прапорець у відповідному рядку, а колонку Нове значення не заповнювати.
  • Натисніть Змінити реквізити.

Результат зміни з помилками

Список незмінених елементів списку виводиться в окремому вікні. У списку по кожному елементу наводиться причина помилки.

  • НатиснітьВідкрити, щоб переглянути елементи, які не вдалося змінити.
  • Натисніть Повторити зміну, щоб спробувати повторно змінити елементи списку з тими ж параметрами.
  • Для виправлення помилки, можливо, доведеться змінити значення реквізитів перерахованих елементів на інші.

Вибір раніше збережених настройок щодо зміни реквізитів

  • Скористайтеся командою Раніше змінені реквізити,
  • З випадаючого списку виберіть потрібну настройку. У списку зберігаються настройки останніх 20 змін реквізитів.

Зміна реквізитів заблокованих

  • У списку можуть бути реквізити, які по умовчанню недоступні для зміни і відображаються сірим кольором.
  • Ці реквізити є "ключовими" реквізитами, вони можуть вплинути на інші списки і документи програми. Тому вони захищені від випадкових змін.
  • При спробі введення нових значень програма пропонує їх розблокувати. У разі особливої необхідності їх зміни, можна підтвердити свій вибір, натиснувши на кнопку Так.

Особливості зміни проведених документів

  • При зміні проведених документів виконується їх повторне проведення. 

Настройка параметрів зміни реквізитів

Програма використовує настройки по умовчанню для зміни реквізитів. Натисніть кнопку Додаткові параметри для того щоб задати настройки.

Групове зміну реквізитів (для адміністратора)

Адміністратору для зміни доступні реквізити всіх довідників і документів програми, в тому числі табличні частини. Також передбачено більш точний вибір елементів для зміни реквізитів (таблиця Вибір елементів для зміни).

Вибір елементів для зміни

  • У полі Змінювати виберіть документ або довідник програми, в якому необхідно змінити елементи. По умовчанню можна Змінювати всі елементи довідника або всі документи обраного виду. 
  • Для перегляду вибраних для зміни елементів скористайтеся гіперпосиланням Змінювати всі елементи.
  • З допомогою команди Додати умову відбору можна відібрати один або кілька елементів за якими-небудь ознаками. При налаштованому відборі зміни проводяться тільки в обраних елементах.
  • Після додавання умови відбору гіперпосилання приймає вигляд Вибрано елементів: і показує, скільки елементів з допомогою відборів вибрано для зміни.
  • Після вибору довідника або документа в таблиці Змінні реквізити можна переглянути доступні для зміни реквізити, у тому числі і службові. Для того щоб показувати службові реквізити, виконайте команду Додаткові параметри та увімкніть прапорець.
  • Після вибору потрібних реквізитів скористайтеся стандартними командами для їх зміни.
