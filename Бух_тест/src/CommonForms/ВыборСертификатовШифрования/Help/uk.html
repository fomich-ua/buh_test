Призначений для вибору сертифікатів електронного підпису (ЕП) при шифруванні файлу.

Відкривається зі списку файлів (приєднаних файлів об'єкта), списку приєднаних файлів або картки файлу (картки приєднаного файлу) по команді ЕП та шифрування - Зашифрувати. Також можна знайти цю команду в меню Всі дії.

У списку виводиться:

  • Вибраний - прапорець;
  • Кому виданий - користувач - власник електронного підпису;
  • Ким виданий - організація, що видала ЕП;
  • Дійсний до - дата дії сертифіката ЕП;
  • Призначення - мета використання сертифіката ЕП, наприклад, "Захищена електронна пошта", "Перевірка автентичності сервера". Для того щоб переглянути детальну інформацію про призначення сертифіката, наведіть мишку на потрібну клітинку.

Шифрування файлу

  • Можуть використовуватися декілька сертифікатів, наприклад, якщо потрібно зашифрувати файл для декількох користувачів.
  • Перед шифруванням необхідно увімкнути прапорець біля сертифікатів користувачів, для яких файл буде зашифровано.
  • Рядки з вибраними сертифікатами стають недоступними, вимкнути прапорці вже неможливо при шифрування всіх файлів списку.
  • Для підтвердження вибору натисніть кнопку Зашифрувати.
  • Якщо файл кілька версій, будуть зашифровані всі версії, для шифрування буде потрібно якийсь час.
    • Перед шифруванням в цьому випадку програма виводить повідомлення. Для підтвердження натисніть Так.  
  • Використані для шифрування сертифікати зазначаються в картці файлу на вкладці Зашифрований для. Команда Зашифрувати після цього стає неактивною.
