Призначена для перегляду відомостей про приєднантй файл. 

Відкривається зі списку приєднаних файлів за допомогою кнопкиВідкрити (клавіші F2).

Відомості про файл

  • Ім'я файлу - повне ім'я файлу. Автоматично заповнюється програмою під час приєднання файлу і створення картки файлу можна змінити;
  • Опис - напишіть докладні відомості про файл;
  • Автор - користувач, який додав файл в базу;
  • Відредагував - останній користувач, який відредагував файл;
  • Розширення - розширення файлу;
  • Редагує - користувач, який зайняв файл для редагування;
  • Створений - дата додавання файлу в базу;
  • Дата зміни - дата зміни файлу даних (місцевий час);
  • Розмір (байт) - розмір файлу даних.

Редагування приєднаного файлу

  • У тонкому клієнті редагування файлів табличних документів (формату .mxl) здійснюється у вбудованій формі редактор табличних документів. У веб-клієнті потрібно завантажити додаток для редагування цих файлів. Детальніше про особливості редагування табличних документів у веб-клієнті див. тут.
  • Для редагування файлів інших форматів на комп'ютері повинні бути встановлені відповідні прикладні програми, наприклад, для редагування файлів у форматі Microsoft Word має бути встановлена ця програма.
  • Команда Відкрити відкриває файл зовнішнім додатком для перегляду або редагування, залежно від того, позначено чи ні файл як зайнятий іншим користувачем для редагування.
  • Команда Редагувати відкриває файл у режимі редагування, а також виконує захоплення файлу в програмі. Захоплений файл буде недоступний для редагування іншим користувачам програми і позначається в списку зеленим кольором.
  • Команда Закінчити редагування звільняє файл, що зберігає його в програмі і знімає позначку зайнятості для редагування. Після закінчення редагування збережіть файл, закрийте зовнішню програму.
  • Команда Всі дії - Скасувати редагування звільняє файл, не зберігаючи вироблені в ньому зміни в програмі, і знімає позначку зайнятості для редагування. Ця команда може бути корисна, якщо необхідно відмовитися від результатів редагування файлу (наприклад, при редагуванні у файлі були внесені небажані зміни або помилки).

Збереження файлу

  • З допомогою команди Всі дії - Зберегти як можна зберегти файл у вказаний каталог на жорсткому або мережевому диску.
  • Відповідною командою меню Всі дії можна Відкрити каталог файлів у Провіднику Windows;

Оновлення файлу

  • З допомогою команди Всі дії - Оновити з файлу на диску можна оновити файл з оригінального файлу на диску (з якого спочатку був створений файл у програмі).

ЕП і шифрування

При використанні ЕП стає доступним випадаюче меню ЕП і шифрування, яке містить команди для роботи з електронним підписом файлів:

  • Підписати - створює ЕП для файлу. Перед створенням ЕП програма пропонує вибрати сертифікат ЕП з наявних на комп'ютері. Вибір сертифіката можна зробити і під час персональних настройок програми. Підписаний файл стає недоступним для редагування і відображається у списках зі значком .
  • Додати ЕП з файлу - додає підпис до файлу. Команду можна використовувати, наприклад, якщо підпис (файл підпису) була отримана пізніше, ніж підписаний файл.
  • Зберегти разом з ЕП - зберігає підпис у файл на диску. Команду можна використовувати, наприклад, якщо необхідно передати підписаний файл іншій особі.
  • Зашифрувати - по команді відбувається шифрування файлу. На відміну від підписання можуть використовуватися декілька сертифікатів, наприклад, якщо потрібно зашифрувати файл для декількох користувачів. Зашифрований файл у списках відображається зі значком .

  • Розшифрувати - по команді відбувається розшифровка файлу. Для розшифровки потрібно ввести пароль закритого ключа.

При використанні ЕП і шифрування стають доступними вкладки:

ЕП

На вкладці розміщуються відомості про електронного підпису файлу:

  • Автор підпису - користувач, який підписав файл;
  • Дата підпису - дата підписання файлу;
  • Коментар - напишіть додаткові відомості про підписи;
  • Статус - істинність підписи.

За допомогою відповідних команд можна:

  • Підписати файл;
  • Відкрити підпис - відкрити вікно з основними властивостями сертифіката ЕП виділеного файлу: кому і ким виданий, призначення і термін дії сертифіката, на основі якого створена підпис;
  • Перевірити, чи змінювався ким-небудь підписаний файл, за допомогою одного або всіх сертифікатів ЕП. Для цього виділіть одну або кілька підписів. Якщо ЕП зберігається файлу не відповідає якому-небудь сертифікату ЕП, то така ЕП після перевірки буде виділено в списку червоним кольором, а в колонці Статус у такий ЕП буде зазначено Не вірна.
  • Перевірити все - перевірити всі підписи;
  • Зберегти ЕП у файл на диск;
  • Вилучити у файла ЕП.

Зашифрований для

  • На вкладці розміщуються відомості про шифрування файлу:
  • У списку відображаються на цій вкладці сертифікати, за допомогою яких зашифрований файл. Подвійним клацанням миші або за допомогою відповідної команди можна Відкрити сертифікат ЕП і переглянути його основні властивості: кому і ким виданий, призначення і термін дії сертифіката.
