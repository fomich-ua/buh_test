Призначена для вибору формату вкладень при відправці друкованих форм по електронній пошті.

Викликається по команді Відправити.

  • Вкажіть за допомогою прапорців, у якому форматі (або декількох форматах) потрібно відправити документ.
  • Для зменшення розміру вкладень друковану форму перед відправкою рекомендується запакувати в архів (формат .zip).
  • Натисніть кнопку Вибрати. Налаштування будуть збережені.
