Призначена для налаштування параметрів основного робочого каталогу.

Основний робочий каталог - каталог на Вашому персональному комп'ютері, в якій тимчасово зберігаються файли, отримані з програми для перегляду або редагування.

Для переходу до настройки параметрів натисніть Настройка основного робочого каталогу у персональних настройках програми (вкладка Робота з файлами).

Розташування основного робочого каталогу

  • У полі Розташування вказується шлях до основного робочого каталогу на Вашому персональному комп'ютері.
  • Встановлюється програмою автоматично, але може бути змінений при необхідності. Для цьогоВиберіть іншу папку на комп'ютері. При виборі іншого шляху до основного робочого каталогу треба вибирати порожній каталог (не містить файлів та підкаталогів).
  • Для того щоб повернутися до вихідних налаштувань програми, натисніть кнопкуВідновити значення по умовчанню.

Максимальний розмір

У полі Максимальний розмір (Мб) встановіть межа сумарного обсягу всіх файлів в основному робочому каталозі (по умовчанню 100 Мб). Якщо встановлений рівним 0, то обсяг файлів не обмежений. Розмір вказується в мегабайтах (Мб).

Запит програми на видалення файлів

Якщо встановлений прапорець Підтверджувати при видаленні з основного робочого каталогу, то перед видаленням кожного файлу буде видаватися запит на видалення файлу. По умовчанню прапорець знятий.

Видалення файлів після редагування

Якщо встановлений прапорець Вилучати файли після завершення редагування, то після перенесення в програму файл вилучається на Вашому персональному комп'ютері. Використовуйте цю настройку для економії місця на жорсткому диску або в цілях підвищення безпеки. По умовчанню прапорець знятий.

Поточний стан

Програма показує Поточний стан основного робочого каталогу - кількість Файлів, які містяться в ньому і їх Загальний обсяг (Мб).

Перегляд списку файлів в основному робочому каталозі

Натисніть Відкрити список файлів, для того щоб переглянути список файлів, які зараз розміщені в основному робочому каталозі.

Очищення основного робочого каталогу

Натисніть Очистити основний робочий каталог, для того щоб вилучити всі файли в основному робочому каталозі, крім зайнятих Вами для редагування. Для підтвердження очищення натисніть Так.
