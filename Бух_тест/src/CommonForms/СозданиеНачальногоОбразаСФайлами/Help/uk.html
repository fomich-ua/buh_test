Призначена для зазначення параметрів файлової або клієнт-серверної бази при створенні початкового образу РІБ. При настройці синхронізації РІБ (зовнішні файли) необхідно вказати, де буде створений початковий образ РІБ. Якщо файли зберігаються в томах, також потрібно буде вказати шлях до файла, в який буде записаний архів (.zip) з усіма файлами в томах.

Відкривається по команді Створити початковий образ із Помічника створення синхронізації даних при настроювання синхронізації РІБ (зовнішні файли). Також можна в списку Синхронізація даних за допомогою кнопки відкрити настройку синхронізації РІБ (зовнішні файли), натиснути кнопку Створення початкового образу підпорядкованого вузла розподіленої інформаційної бази.

Параметри інформаційної бази

Вкажіть за допомогою перемикача, де буде створюватися початковий образ інформаційної бази:

  • На даному комп'ютері або на комп'ютері в локальній мережі.
    • Задайте Параметри інформаційної бази:
    • У полі Мова (Країна) виберіть потрібний параметр зі списку. По умовчанню пропонуються параметри, які були задані при створенні синхронізації даних.
    • Вкажіть Повне ім'я файлової бази:
      • Для сервера 1С:Підприємства під управлінням Microsoft Windows - за допомогою кнопки вкажіть шлях до папки, в якій буде розміщений файл.

або

      • Для сервера 1С:Підприємства під управлінням Linux - напишіть шлях до папки, в якій буде розміщений файл початкового образу.
  • На сервері 1С:Підприємства.
    • Задайте Параметри інформаційної бази:
      • Вкажіть Кластер серверів 1С:Підприємства;
      • Напишіть Ім'я інформаційної бази в кластері;
      • У полі Захищене з'єднання виберіть із списку (Вимкнено, Встановлення з'єднання, Постійно);
      • Виберіть зі списку Тип СУБД (Microcoft SQL Server, PostgreSQL, IBM DB2, Oracle Database);
      • Вкажіть Сервер бази даних;
      • Напишіть Ім'я бази даних;
      • Напишіть ім'я Користувача бази даних, Пароль Користувача;
      • Виберіть Зміщення дат (по умовчанню 0);
      • У полі Мова (Країна) виберіть параметр зі списку. По умовчанню пропонуються параметри, які були задані при створенні синхронізації даних;
      • При необхідності увімкніть прапорець Встановити блокування регламентних завдань (по умовчанню вимкнено).

Шлях до архіву з файлами томів

Вкажіть Повне ім'я архіву з файлами томів:

  • Для сервера 1С:Підприємства під управлінням Microsoft Windows - вкажіть за допомогою кнопки шлях до каталогу

або

  • Для сервера 1С:Підприємства під управлінням Linux - напишіть шлях до папки, в якій буде розміщений архів з файлами томів.

Для продовження натисніть Створити початковий образ.

Через деякий час програма виводить повідомлення про успішне створення початкового образу підпорядкованого вузла розподіленої інформаційної бази. Натисніть ОК для повернення в програму.

Для того щоб завантажити файли томах в базі філії, при першому запуску необхідно виконати команду Розмістити файли початкового образу панелі дій в розділі Органайзер, вказавши шлях до файлу архіву з файлами в томах.
