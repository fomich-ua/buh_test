Призначена для перегляду, редагування та друку сформованих друкованих форм, а також для відправки друкованої форми по електронній пошті і збереження у файл.

Викликається з документів і списків документів програми по командіДрук. 

Передбачені наступні можливості:

  • Друк документів - загальна друкарська форма для всіх документів.
  • Друк комплекту;
  • Настроюваний комплект.

Висновок на принтер

Натисніть Друк для виведення потрібної кількості копій сформованої друкованої форми на принтер.

Кількість копій

Вкажіть кількість Копій. Обрана кількість копій запам'ятовується. Для комплектів документів кількість копій залежить від налаштувань комплекту.

Редагування друкованої форми

З допомогою кнопки Редагування можна включити режим редагування цієї друкованої форми. У цьому випадку друкована форма буде зайнята і інші користувачі не зможуть її редагувати. Повторним натисканням кнопки можна вимкнути режим редагування.

Збереження друкованих форм

По команді Зберегти є можливість зберегти друковані форми в потрібних форматах в папку на комп'ютері або приєднати до документа. Виберіть за допомогою перемикача спосіб збереження документа і потрібні формати для збереження.

Відправка по електронній пошті

Сформовані друковані форми документа можна Відправити в якості вкладень в електронному листі.

  • Перед відправкою необхідно вказати за допомогою прапорців, у якому форматі (або декількох форматах) потрібно відправити документ.
  • Для зменшення розміру вкладень друковану форму перед відправкою рекомендується запакувати в архів (формат .zip). Натисніть кнопку Вибрати. Налаштування будуть збережені.
  • Заповніть поля повідомлення Кому, Тема, напишіть зміст листа. Натисніть кнопку Надіслати.
  • При відправці комплекту документів для кожної друкованої форми буде сформовано окремий вкладення. Якщо друкується комплект друкарських форм відразу для декількох документів зі списку, то рекомендується розбивати документи по контрагентам. У цьому випадку зручно відправляти весь комплект одному контрагенту, наприклад, якщо в комплект входить дві друковані форми, то в першому файлі будуть всі рахунки, у другому всі накладні.

Перехід до потрібного документу

  • Зі списку документів для друку можна вибрати кілька документів.
  • У цьому випадку буде виведено кілька друкованих форм. Для швидкого переходу до потрібної друкованій формі натиснітьПерейти до документа (клавішу F7).
  • Із запропонованого списку виберіть потрібний документ, натисніть ОК.

Редагування макету

Натисніть Відкрити макет для редагування, для того щоб перейти до редагування макету друкованої форми. Друкована форма стає неактивною. Після завершення редагування друкована форма зміниться автоматично.

Друк комплекту

Програма дозволяє роздрукувати комплект різних документів. Друковані форми комплекту виводяться на окремих сторінках, при цьому можна переглянути конкретну форму, вибираючи з списку друкованих форм зліва потрібну. Можна змінити кількість Копій комплекту.

Настроюваний комплект

У програмі передбачена можливість вибіркової друку комплекту документів.

  • У вікні Комплект, що настроюється зліва знаходиться панель настройки кількості копій, яка складається з стовпців:
    •   - в цьому стовпці за допомогою прапорців проводиться вибір для друку 
    • Друкована форма;
    • кількість Копій.
  • Для того щоб налаштувати друк комплекту, використовуйте кнопку Настройка комплекту. З допомогою команд контекстного меню настройки можна Встановити або Зняти всі прапорці, поміняти порядок друкованих форм в комплекті (Перемістити вгору/вниз) або Скинути налаштування комплекту.
  • Виберіть потрібну друковану форму з комплекту для перегляду. Увімкніть прапорець для друку цієї друкованої форми. Проставте необхідну кількість копій для кожної друкованої форми. Налаштування будуть збережені.
  • Крім цього, можна змінити кількість Копій комплекту.
  • Для зручності перегляду друкованої форми можна відключити панель настройки кількості копій з допомогою кнопки Показати/Приховати настройку кількості копій.
