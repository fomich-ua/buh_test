Звіт Структура підпорядкованості призначений для виведення батьківських і підпорядкованих документів. Документ, для якого сформовано звіт відображається жирним шрифтом. Дерево батьківських документів будується вгору-вправо від документа, для якого формується звіт. Дерево дочірніх документів будується вниз-вправо від документа, для якого формується звіт.

За допомогою подвійного клацання на будь-якому виведеному в звіт документі можна відкрити документ для перегляду.

Скориставшись кнопкою Вивести для поточного можна сформувати звіт Структура підпорядкованості для документа, на якому ми на даний момент зпозиціонувалися.

Для того, щоб оновити звіт використовуйте кнопку Оновити.
