У формі виконуються персональні настройки електронного підпису (ЕП).

Настройка Опис
Дії при збереженні з ЕП

Задає поведінку при виконанні команди "Зберегти з ЕП":

  • Зберігати всі підписи - всі підписи зберігаються;
  • Запитувати - вибір зберігаються підписів здійснюється у формі.
Розширення для зашифрованих файлів Задає розширення для зашифрованих файлів при відправці поштою, при виконанні команди "Зберегти як" і при експорті папки.
По умовчанню "p7m".
Шлях модуля криптографії (для веб клієнта під управлінням Linux) Задає шлях модуля криптографії для операцій з файлами (розшифровка, підписання, перевірка підпису, перегляд сертифіката).
Особистий сертифікат для шифрування

Персональний сертифікат електронного підпису (ЕП) для шифрування даних. При виборі сертифікатів для шифрування даний сертифікат буде завжди відзначений. Це потрібно для того, щоб не можна було зашифрувати дані без можливості їх розшифровки.
