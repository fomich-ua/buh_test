Призначений для роботи з файлами, приєднаними до об'єкту програми.

Відкривається по команді Приєднані файли панелі навігації об'єкта програми.

У списку виводиться:

  •  - підписаний ЕП.
  •  - зашифрований.
  • Найменування - повне ім'я файлу. Значок поруч з найменуванням вказує на тип файлу.
  • Опис - докладні відомості про файл.
  • Дата зміни - дата зміни файлу даних (місцевий час);
  • Автор - користувач, який додав файл;
  • Розмір (КБ) - розмір файлу даних;

З допомогою команди Всі дії - Змінити форму можна додати відомості:

  • Розширення - формат файлу, поруч ставиться значок, який також показує формат;
  • Редагує - користувач, який зайняв файл для редагування і т.д. 

Додавання нового файлу

  • Новий файл можна Додати в список з вже наявного файлу на диску.
  • Після вибору файлу на диску автоматично відкривається вікно Приєднаний файл, що описує властивості файлу.
  • Файл зберігається в спеціальному каталозі. Для кожного об'єкта програми формується окрема папка.
  • В деяких списках елементи, до яких приєднані файли, позначаються піктограмою .

Перегляд картки файлу

  • Натисніть ( клавішу F2) , для того щоб переглянути властивості виділеного в списку файлу, відкривши вікно Приєднаний файл, також можна скористатися командою Всі дії - Властивості файлу.

Збереження файлу

  • За допомогою кнопки можна зберегти файл у вказаний каталог на жорсткому або мережевому диску.
  • Також можна скористатися командою Всі дії - Зберегти як.
  • Відповідною командою меню Всі дії можна Відкрити каталог файлів у Провіднику Windows;

Оновлення файлу

  • З допомогою команди Всі дії - Оновити з файлу на диску можна оновити файл з оригінального файлу на диску (з якого спочатку був створений файл у програмі).

Редагування приєднаних файлів

  • У тонкому клієнті редагування файлів табличних документів (формату .mxl) здійснюється у вбудованій формі редактор табличних документів. У веб-клієнті потрібно завантажити додаток для редагування цих файлів. Детальніше про особливості редагування табличних документів у веб-клієнті див. тут.
  • Для редагування файлів інших форматів на комп'ютері повинні бути встановлені відповідні прикладні програми, наприклад, для редагування файлів у форматі Microsoft Word має бути встановлена ця програма.
  • Команда Відкрити відкриває файл зовнішнім додатком для перегляду або редагування, залежно від того, позначено чи ні файл як зайнятий іншим користувачем для редагування.
  • Команда Редагувати відкриває файл у режимі редагування, а також виконує захоплення файлу в програмі. Захоплений файл буде недоступний для редагування іншим користувачам програми і позначається в списку зеленим кольором.
  • Команда Закінчити редагування звільняє файл, що зберігає його в програмі і знімає позначку зайнятості для редагування. Після закінчення редагування збережіть файл, закрийте зовнішню програму.
  • Команда Всі дії - Скасувати редагування звільняє файл, не зберігаючи вироблені в ньому зміни в програмі, і знімає позначку зайнятості для редагування. Команда може бути корисна, якщо необхідно відмовитися від результатів редагування файлу (наприклад, при редагуванні у файлі були внесені небажані зміни або помилки).

ЕП і шифрування

У випадаючому меню ЕП і шифрування доступні команди для роботи з електронним підписом файлів:

  • Підписати - створює ЕП для файлу. Перед створенням ЕП програма пропонує вибрати сертифікат ЕП з наявних на комп'ютері. Вибір сертифіката можна зробити і під час персональних настройок програми. Підписаний файл стає недоступним для редагування і відображається у списках зі значком .

  • Додати ЕП з файлу - додає підпис до файлу. Команду можна використовувати, наприклад, якщо підпис (файл підпису) була отримана пізніше, ніж підписаний файл.

  • Зберегти разом з ЕП - зберігає підпис у файл на диску. Команду можна використовувати, наприклад, якщо необхідно передати підписаний файл іншій особі.

  • Зашифрувати - по команді відбувається шифрування файлу. На відміну від підписання можуть використовуватися декілька сертифікатів, наприклад, якщо потрібно зашифрувати файл для декількох користувачів. Зашифрований файл у списках відображається зі значком .

  • Розшифрувати - по команді відбувається розшифровка файлу. Для розшифровки потрібно ввести пароль закритого ключа.
