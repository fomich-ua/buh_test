Призначений для перегляду і введення значень додаткових відомостей. Введення значень додаткових відомостей, як правило, здійснюється в окремому вікні, на відміну від додаткових реквізитів.

Відкривається по команді Додаткові відомості з вікон об'єктів програми, для яких були настроєні додаткові відомості.

У списку виводиться:

  • Найменування - найменування поля додаткового відомості;
  • Поле Значення призначене для перегляду чи вводу значень додаткової відомості.

Введення значень

  • Для переходу в режим введення або редагування значення додаткового відомості можна двічі клацнути мишею по полям.
  • Спосіб введення значення залежить від типу додаткового відомості (детальніше про типи додаткових реквізитів і відомостей див. тут).
  • Додаткові відомості можуть мати вибір лише з двох значень, наприклад, Да чи Ні. У цьому випадку введення нових значень не передбачено.
  • Якщо для введення передбачений список значень, натисніть Показати всі (клавішу F4). Потім виберіть потрібне значення із запропонованого списку. В залежності від настройок список значень може бути з ієрархією або без ієрархії. При необхідності в ці списки можна додавати значення.
  • Вибір значень (в залежності від налаштувань) може здійснюватися також з довідників програми.
  • В залежності від настройок додаткових відомостей може знадобитисяВибрати тип даних.

Зміна складу додаткових відомостей

При наявності відповідних прав можна настроїти нове додаткове зведення або змінити деякі налаштування вже існуючих додаткових відомостей. Для цього виконайте команду Всі дії (в інтерфейсі Таксі Ще) - Змінити склад додаткових відомостей.

Вилучені додаткові відомості

Додаткова відомость, яку було відмічено на вилучення, зображується зі значком . Для того щоб остаточно вилучити додаткову відомость, можливо, доведетьсяОчистити значення. Двічі клацніть мишею по значенню, для того щоб перейти в режим редагування.
