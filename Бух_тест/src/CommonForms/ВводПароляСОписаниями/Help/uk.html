Призначена для введення пароля при розшифровці файлу.

Відкривається по команді ЕП та шифрування - Розшифрувати (Всі дії - ЕП та шифрування - Розшифрувати) у списку файлів (приєднаних файлів), списку приєднаних файлів або картці файлу (картці приєднаного файлу).

Відображаються відомості:

  • У поле Файл вказується найменування файлу, недоступне для редагування.
  • У поле Файл зашифрований для вказується, для яких персональних сертифікатів електронного підпису (ЕП) зашифрований файл, недоступний для редагування.

    Розшифровка файлу

    • У полі Пароль введіть пароль для розшифровки файлів, для підтвердження натисніть OK.

    Див. також:

    • Налаштування ЕП та шифрування;
    • Персональні настройки ЕП.
